# Reference Links

## Chinese Stroke Order Repositories

| Repo | Description |
|------|-------------|
| [skishore/makemeahanzi](https://github.com/skishore/makemeahanzi) | Free, open-source data for 9,000+ simplified/traditional characters with stroke-order SVGs. The most widely used underlying dataset. |
| [chanind/hanzi-writer](https://github.com/chanind/hanzi-writer) | JavaScript library for stroke order animations and practice quizzes (built on Make Me a Hanzi data). |
| [parsimonhi/animCJK](https://github.com/parsimonhi/animCJK) | SVG stroke-by-stroke animations for ~7,600 simplified hanzi, plus kanji/kana/hanja. |
| [Connum/hanzivg](https://github.com/Connum/hanzivg) | SVG stroke order files with radical/component metadata (the "KanjiVG for Chinese"). |
| [chill-chinese/stroke-order-animator](https://github.com/chill-chinese/stroke-order-animator) | Flutter package for stroke order animations/quizzes. |
| [Eloston/chinese-stroke-order](https://github.com/Eloston/chinese-stroke-order) | Online stroke order lookup tool. |
| [yindian/hzbishun](https://github.com/yindian/hzbishun) | Raw stroke order (bishun) data for hanzi. |
